Service account:
Project:

Upload a service account JSON key from your GCP project to enable Google Chirp 2. See the setup guide for which IAM role to grant (Cloud Speech Client) and where to download the key.

Mic auto-stops after 1 hour while Google Chirp is active (~$3/hr at our cadence). The key file is stored locally under your app-data folder, not transmitted to us.

Advanced — tracking thresholds

Per-session overrides for the line-tracking knobs. Useful for tuning line-flicker (raise hysteresis) or context length (raise window). Bounds are enforced server-side.

⚠ Google Chirp is active. Changing these knobs will affect cost. The local model has no such concern.

Type first letter of each word, e.g. ssam

This is not a fully working app. It is a prototype to gather feedback & share ideas. For example, it struggles with faster Kirtan. Please forgive any mistakes.

How to improve results:

  • Plug sound system output directly into computer, or move computer closer to speaker
  • Try a different speech-to-text model. The local model runs on your computer for free, but is small. The Google model is paid but has better accuracy
  • See the code yourself here and take anything useful

Please do not leave fully unattended in Sangat (you can override the shabad & line manually by stopping mic) 🙏
